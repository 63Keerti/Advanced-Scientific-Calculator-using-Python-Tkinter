This project is an Advanced Scientific Calculator built using Python and Tkinter. It supports both basic arithmetic and scientific functions with a modern GUI, dark/light theme, history tracking, and keyboard support.

🚀 Features
➕ Basic operations: Add, Subtract, Multiply, Divide
📐 Scientific functions: sin, cos, tan, log, sqrt, power, modulus
🌙 Dark / ☀️ Light mode toggle
📜 Calculation history (saved in file)
⌨️ Keyboard input support
🧠 Error handling for invalid inputs
💾 Saves history using JSON file
🎨 Clean and responsive GUI
🛠 Requirements

Make sure you have Python installed:

python --version

No extra libraries required (only built-in modules):

tkinter
math
json
os

Enter numbers using buttons or keyboard
Choose operations or scientific functions
Press = to calculate result
View past calculations in history panel
Switch between dark and light themes
